# Realm Forge

A modern gaming site for **William S.** — built as a static website for GitHub Pages.

## Publish on GitHub Pages
Repo Settings → Pages:
- Source: Deploy from a branch
- Branch: main
- Folder: /(root)

Live URL (for your repo name `realmforge`):
- https://williamsethi.github.io/realmforge/

## Notes
- Thumbnails are in `assets/img/` (SVG illustrations)
- Icons/hero are in `assets/svg/`
