# Realm Forge

A simple, modern gaming site for **William S.** — built as a static website for GitHub Pages.

## Files
- `index.html` (Home)
- `about.html`
- `games.html`
- `reviews.html`
- `videos.html`
- `style.css`
- `main.js`
- `assets/svg/*` (original icons + hero background)

## Publish on GitHub Pages (no credit card)
1. Create a new GitHub repository named:
   - **Option A (recommended):** `realm-forge` (project site)
   - **Option B:** `<your-username>.github.io` (your main site)

2. Upload **all files** in this folder to the repo root (same level as `index.html`).

3. In the repo: **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/(root)**

4. Your site will be live at:
   - Project site: `https://<your-username>.github.io/realm-forge/`
   - Main site: `https://<your-username>.github.io/`

## Customize
- Change the game list: edit `games.html`
- Add new reviews: edit `reviews.html`
- Add real YouTube IDs: edit `videos.html`
